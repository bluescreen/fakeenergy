Hi,

ich wollte auf eurer Webseite mal schauen ob ihr meine Frage in
den FAQ beantwortet (zur Kündigungsfrist bei Wechsel) — aber
sobald ich auf irgendeine Frage klicke, öffnen sich alle anderen
Fragen gleichzeitig. Das macht das Lesen unmöglich, ich kann nicht
gezielt EINE Antwort sehen ohne dass die ganze Seite explodiert.

Ich habe es in drei Browsern getestet:
- Chrome auf MacOS
- Safari auf MacOS
- Firefox auf MacOS

Überall das gleiche Verhalten. Auf dem iPhone (Safari mobile) auch
genau so.

Liegt das vielleicht an einem CSS-Bug? Mein Cousin hat mir mal
erklärt dass HTML "details/summary" eigentlich genau dafür gemacht
ist, dass nur eine Antwort offen ist. Vielleicht habt ihr das
deaktiviert.

Oder ist das Absicht und ich bin der einzige der das nervig findet?

Ist auf jeden Fall ein User-Experience-Problem. Bei euren Mit-
bewerbern (E.ON, Vattenfall) klappt das mit den FAQ einwandfrei.

Grüße
M. Wahl
