Hi,

ich habe gestern euren Newsletter mit dem Code WELCOME25 bekommen
('25 Euro Bonus für Neukund:innen, jetzt sichern!'). Hab mich gleich
registriert weil ich eh wechseln wollte.

Beim Funnel den Code eingegeben so wie er bei mir im Newsletter
stand — also "welcome25" weil ich den Code aus der Mail kopiert hab
und der war da klein geschrieben (komisch, eure Headline ist groß
aber der eigentliche Code im Fließtext ist klein). Hab mir nichts
weiter dabei gedacht, hab einfach den Code per Copy/Paste eingefügt.

Auf der Empfehlungsseite hat es funktioniert. Stand groß da:
"Bonus 25 EUR". Ich war zufrieden und hab Vertrag abgeschlossen.

Jetzt kommt die erste Rechnung und der 25 EUR Rabatt fehlt KOMPLETT.
Stattdessen steht ganz unten in einem winzigen Hinweistext:
"unknown_promo_code:welcome25". Was soll das?? Ich habe genau den
Code eingegeben den IHR mir geschickt habt.

Ich habe es nochmal mit nem Test-Account probiert (mit anderer
E-Mail, hab ne Wegwerf-Mail genommen) und WELCOME25 in groß
eingetippt — funktioniert da. Wenn ich klein tippe — geht nicht.
Das kann doch nicht euer Ernst sein.

Mein Vater hat das gleiche Problem, er hat auch klein getippt. Bei
ihm ist es noch schlimmer weil er eine Augenerkrankung hat und mit
dem Spracheingabe-Feature seines iPhones eingibt — die schreibt
grundsätzlich klein.

Ist das eine Falle damit ihr den Bonus nicht zahlen müsst? Wenn ja
sehr unsauber. Wenn nicht bitte korrigiert das.

**Auf dem Newsletter steht der Code übrigens an einer Stelle GROSS
(WELCOME25 in der Betreffzeile) und an zwei anderen Stellen klein
(welcome25 im Fließtext). Wenn man auf einem Smartphone draufklickt
übernimmt der Browser das so wie er es findet, und das ist halt
klein.** Vielleicht solltet ihr da konsistent sein.

Ich erwarte 25 EUR auf der nächsten Rechnung. Sonst Rückbuchung
und Wechsel zu Vattenfall.

Grüße
J. Petrov
